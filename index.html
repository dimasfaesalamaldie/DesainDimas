About Me

About Me

Hii, saya seorang UI & UX Designer yang memiliki latar belakang lulusan Teknik Informatika serta memiliki pengalaman untuk merancang User Interface dan User Experience selama 3 tahun. Dalam merancang User Interface saya mempertimbangkan komponen berdasarkan dasar-dasar hukum UI & UX untuk mempertimbangkan Experience user agar user dapat menggunakan aplikasi mobile ataupun website dengan nyaman. Dengan visi yang kuat untuk menciptakan produk, saya berlatih mengembangkan diri dengan mengikuti pelatihan UI & UX ataupun mengasah skil dengan cara membuat tampilan-tampilan yang baru. Saya adalah orang yang cepat belajar, dan memiliki kemampuan desain kritik. Berada dalam situasi yang melibatkan banyak tim, membuat saya mampu bekerja sama dengan melibatkan tim, mulai dari brainstorming, wireframing, high fiddlety hingga menggunakan prototype untuk mempresentasikan pada CEO ataupun menggunakanya untuk user research.
